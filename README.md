<h1 align="center">Lasso</h1>

<p align="center">A lightweight macOS focus router for long-running work.</p>

---

Lasso is an early personal macOS prototype for managing attention during wait-heavy workflows.

The public repo is intentionally minimal while the project is still experimental. Local automation adapters, development notes, and machine-specific setup files are kept private on the developer machine.
